# LILYGO® T-Beam S3-Core

The T-Beam S3-Core is the supported hardware platform for the Drone Detector project.

## [T-Beam S3-Core](./tbeam/)

| Component | Specification |
|-----------|---------------|
| MCU | ESP32-S3 (WiFi & Bluetooth 5 LE) |
| Radio | Semtech SX1262 |
| Frequencies | 433 MHz, 868 MHz, or 915 MHz |
| GPS | NEO-M10S or Quectel L76K |
| Interface | USB-C, Boot/Reset/Power buttons |

### Key Features

- **SX1262 LoRa Transceiver** - Excellent sensitivity for signal detection
- **ESP32-S3 MCU** - Powerful dual-core processor
- **Integrated GPS** - Log coordinates of detected signals
- **Low power** - Suitable for portable/field use

For detailed specifications and button controls, see the [T-Beam documentation](./tbeam/).
