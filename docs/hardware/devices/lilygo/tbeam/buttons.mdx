# Button Controls

The T-Beam S3-Core features three physical buttons for device control and menu navigation.

## Button Layout

| Button | Position | Primary Function |
|--------|----------|------------------|
| **User/Boot** | Left | Menu navigation and selection |
| **Power** | Middle | Power on/off |
| **Reset** | Right | Device restart |

## Button Functions

### Power Button (Middle)

| Action | Result |
|--------|--------|
| Long press (3s) | Power on the device |
| Long press (3s) while on | Power off the device |

### Reset Button (Right)

| Action | Result |
|--------|--------|
| Single press | Restart the device |

### User Button (Left)

The User button is the primary control for the Drone Detector interface:

| Action | Result |
|--------|--------|
| Single press | Navigate to next menu item |
| Double press | Select / Confirm current option |
| Long press (3s) | Start or stop RF scanning |
| Triple press | Switch between frequency bands |

## Menu Navigation

Use the User button to navigate the Drone Detector menus:

1. **Single press** to move through menu options
2. **Double press** to select the highlighted option
3. **Long press** to return to scanning mode from any menu

## Scanning Controls

While in scanning mode:

| Action | Result |
|--------|--------|
| Single press | View detection details |
| Double press | Mark/flag current detection |
| Long press | Stop scanning, return to menu |
| Triple press | Quick-switch frequency band |

## Boot Mode

To enter firmware flashing (boot) mode:

1. Power off the device
2. Press and hold the **User/Boot** button
3. While holding, plug in USB-C
4. Release after 2-3 seconds

The device is now in ESP32-S3 firmware download mode.
