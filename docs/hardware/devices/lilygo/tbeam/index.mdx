# T-Beam S3-Core

The LILYGO T-Beam S3-Core is the primary hardware platform for the Drone Detector project.

## Specifications

| Component | Specification |
|-----------|---------------|
| **MCU** | ESP32-S3 (WiFi & Bluetooth 5 LE) |
| **LoRa Transceiver** | Semtech SX1262 |
| **Frequency Options** | 433 MHz, 868 MHz, or 915 MHz |
| **GPS Module** | NEO-M10S or Quectel L76K |
| **Connector** | USB-C |
| **Antenna** | U.FL connector |

## Features for Drone Detection

- **SX1262 Radio** - Excellent sensitivity down to -148 dBm
- **Wide frequency range** - 150 MHz to 960 MHz capability
- **Multiple modulation modes** - LoRa, FSK, GFSK, MSK, OOK
- **Channel Activity Detection (CAD)** - Hardware-accelerated signal detection
- **RSSI measurement** - 1 dB resolution for signal strength
- **GPS integration** - Log location of detected signals

## Buttons

| Button | Location | Function |
|--------|----------|----------|
| **Boot** | Left | User button for menu navigation |
| **Power** | Middle | Long press to power on/off |
| **Reset** | Right | Single press to restart |

### Button Actions for Drone Detector

| Action | Function |
|--------|----------|
| Single press (User) | Navigate menu / Next item |
| Double press (User) | Select / Confirm |
| Long press (User) | Start/Stop scanning |
| Triple press (User) | Switch frequency band |

## Flashing Firmware

To flash the Drone Detector firmware:

1. Unplug the device
2. Press and hold the **BOOT** button
3. Plug in the USB-C cable while holding BOOT
4. Release BOOT after 2-3 seconds
5. The device is now in firmware download mode
6. Flash using ESP-IDF, PlatformIO, or Arduino IDE

:::warning
Always connect an antenna before powering on to avoid damaging the radio!
:::
